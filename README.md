# correio-da-pos-verdade
O **Correio da Pós-Verdade** é um agregador de notícias que reúne em uma única página os principais despachos de veículos independentes, investigativos e progressistas do Brasil. A interface imita a estética dos jornais impressos da década de 1930, tipografia serifada, colunas, capitulares e papel envelhecido.
